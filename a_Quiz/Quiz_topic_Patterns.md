## Q8. Inverse data flow‚ùå

Q8 ‚Äî Inverse data flow = Child calls a Parent callback to update Parent's existing state. Different from lifting state up which is a refactoring action.

### Answer

- **Lifting state up** means you move state that was in a Child component up to the Parent ‚Äî it's a refactoring action.
- **Inverse data flow** means the state already lives in Parent, and the Child calls a callback to update it ‚Äî data flows upward from Child to Parent.
